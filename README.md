A new Flutter project.

## tree

jsons                 
├─ drinkingInfo.json  
├─ profile.json       
└─ userInfo.json      

lib                                  
├─ common                            
│  ├─ Global.dart                    
│  └─ UserModel.dart                 
├─ models                                                     
├─ routes                
│  └─ home_page.dart                 
├─ states                                            
├─ widgets                           
│  └─ Button.dart                    
└─ main.dart                         



| 文件夹  | 作用                                                         |
| ------- | ------------------------------------------------------------ |
| common  | 一些工具类，如通用方法类、保存全局变量的静态类 |
| models  | Json文件对应的Dart Model类会在此目录下                       |
| states  | 保存APP中需要跨组件共享的状态类                              |
| routes  | 存放所有路由页面类                                           |
| widgets | APP内封装的一些Widget组件都在该目录下                        |
